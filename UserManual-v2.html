How to Download Videos

Step-by-Step Download Process

  1. Copy Video URL

    Navigate to the video you want to download, copy its URL from your browser's address bar.

  2. Paste URL into Voltrim

    Click in the "Video URL" field and paste (Ctrl+V) the copied URL.

  3. Select Quality

    Choose your desired quality from the dropdown menu (Best Quality, 1080p MP4, 720p MP4, or Audio MP3).

  4. (Optional) Change Save Location

    Click "CHANGE FOLDER" if you want to save to a different location.

  5. Duplicate Detection Check

    Click the blue "DOWNLOAD" button. Voltrim will now check if the file already exists:

    • If file exists: A dialog appears showing file details and asking what to do:
      • YES: Skip download and keep existing file
      • NO: Delete old file and download fresh
      • CANCEL: Cancel this download
    • If file doesn't exist: Download starts immediately
  6. Monitor Progress

    Watch the progress bar and log window for real-time updates. The status message will show percentage complete.

  7. Download Completes

    When done, you'll see a green success panel with options to "Open Folder" or start a "New Download".

Troubleshooting

Download Fails or Errors

‚ö†Ô∏è

Common Solutions:

  • Update Components: Click "CHECK UPDATES" to get latest yt-dlp version
  • Check URL: Ensure the URL is correct and the video is still available
  • Platform Changes: Some platforms change their systems - updates usually fix this
  • Network Issues: Check your internet connection
  • Age Restrictions: Some videos may require authentication

Slow Download Speed

  • Check your internet connection speed
  • Close other bandwidth-intensive applications
  • Try downloading at different times
  • Source platform's server speed may limit downloads

Application Won't Start

  • Ensure you're running Windows 10/11 64-bit
  • Check antivirus hasn't quarantined the application
  • Run as administrator if needed
  • Verify .NET Framework is installed (usually pre-installed on Windows 10/11)

Video Quality Lower Than Expected

  • Not all videos are available in all quality options
  • Platform may have removed higher quality versions
  • Try "Best Quality" option for highest available
  • Check source video quality on the platform

Best Practices for Duplicate Detection

  1. Check File Info: When the duplicate dialog appears, review the file size and date to ensure it's the version you want
  2. Choose Overwrite for Updates: If the original video was updated on the platform, choose NO (Overwrite) to get the latest version
  3. Choose Skip for Bandwidth: If the file is correct, choose YES (Skip) to save time and bandwidth
  4. Different Quality = New File: Changing quality settings will create a new file (e.g., 1080p vs 720p are separate files)

Best Practices for Legal Compliance

  1. Know the Platform Restrictions: YouTube, Instagram, TikTok, Netflix, Hulu, Disney+, Facebook, and Twitter/X all PROHIBIT downloading. Assume prohibition unless explicitly stated otherwise.
  2. Document Permissions: Keep records of any permissions you receive to download content (emails, licenses, written agreements)
  3. Read Terms Carefully: Platform Terms of Service can change - review them regularly before downloading
  4. When in Doubt, Don't Download: If you're unsure about permissions, ask for clarification first or seek legal advice
  5. Respect Copyright: Remember that copyright law protects creators' work regardless of technical capability
  6. Personal Use Only: Unless you have explicit distribution rights, keep downloads private and never redistribute
  7. Check Licenses: If downloading Creative Commons content, verify the specific license terms (some prohibit commercial use, require attribution, etc.)
  8. Your Content First: The safest downloads are your own uploads - focus on backing up content you've created
  9. Understand Consequences: Violations can result in account bans, legal action, lawsuits, and even criminal prosecution in severe cases
